
## 四、交互规范

### 4.1 列表操作
- 支持点击表头排序
- 筛选条件即时生效
- 分页保持筛选状态

### 4.2 Modal操作
- 点击遮罩关闭（可配置）
- ESC键关闭
- 内容实时保存草稿

### 4.3 流程节点
- 成功节点：绿色，已填写内容可查看不可编辑
- 进行中节点：蓝色，可编辑
- 失败节点：红色，显示拒绝原因，可编辑

### 4.4 通知
- 流程扭转自动飞书通知
- 审核结果通知带链接

## 五、响应式设计

### 5.1 断点
```
xs: < 576px  (手机)
sm: 576px-768px  (平板)
md: 768px-1200px  (小屏笔记本)
lg: 1200px-1600px (桌面)
xl: > 1600px (大屏)
```

### 5.2 布局适配
- 看板页：单列 → 双列 → 三列
- 应用卡片：2列 → 3列 → 4列
- 代办区域：可收起
