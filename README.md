# 独立三方应用发布系统

三方应用发布管理平台，支持应用申请、审核、物料上传、灰度发布全流程。

## 在线地址

**生产环境**: https://frontend-wine-one-87.vercel.app

## 功能模块

- ✅ 通道发布申请
- ✅ 通道发布审核
- ✅ 物料上传
- ✅ 物料审核
- ✅ 应用上架
- ✅ 业务内测
- ✅ 灰度监控

## 技术栈

- 前端: React + TypeScript + Tailwind CSS + Vite
- 后端: Express + JSON 数据库

## 部署

通过 Vercel 自动部署，代码推送 GitHub 后自动触发部署。

## 开发

```bash
# 前端
cd frontend && npm install && npm run dev

# 后端
cd backend && npm install && npm run dev
```
